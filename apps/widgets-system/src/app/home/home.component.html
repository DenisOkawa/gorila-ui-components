Getting started

1. Install the library

$ npm install --save @gorilainvest/ui-toolkit

$ yarn add @gorilainvest/ui-toolkit

2. Update your tsconfig paths

...
  "paths": {
    ...
    "@gorilainvest/ui-toolkit/*": [
      "node_modules/@gorilainvest/ui-toolkit/dist/*"
    ]
  }
...
